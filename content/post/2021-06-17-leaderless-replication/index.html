introduction

The typical approach for replication is a leader-follower system, where the leader broadcast changes to the followers, like the diagram below.

However, another style is also used in systems like Cassandra and Riak, where clients send requests to server nodes concurrently, just like the diagram below. There are no difference between leaders and followers in such systems, so they are called leaderless replication. they are also called dynamo-style system because the first system introducing such style is dynamo from Amazon.
